# PotterPlus Configuration for the Magic Plugin

PotterPlus is an expansive, lore-friendly Harry Potter configuration for [NathanWolf's Magic plugin](https://www.spigotmc.org/resources/magic.1056/) which is derived from Magic's [original Harry Potter configs](https://github.com/elBukkit/MagicPlugin/wiki/Potter).
